AloudGuide

Privacy Policy

Last updated: June 21, 2026

AloudGuide is a browser extension that reads a short, plain-English explanation of the web page you are viewing when you ask it to.

What AloudGuide Collects

When you click the AloudGuide button, the extension creates a small page descriptor. It can include:

  • The site's title, hostname, and public page description.
  • Headings and the labels of prominent buttons.
  • Form field labels and whether each field is empty or filled.
  • Page state signals, such as cart count, visible error messages, or whether a confirmation message appears.
  • A local safety verdict about whether the web address looks suspicious.

AloudGuide does not collect raw HTML. AloudGuide does not collect typed form values, passwords, payment card numbers, or other field contents. The extension may look at a field value only long enough to decide whether the field is empty or filled, then discards the value.

How The Data Is Used

When you ask AloudGuide to read a page, the page descriptor is sent over an encrypted (HTTPS) connection to the AloudGuide backend service. The backend forwards the descriptor to Anthropic's Claude API, which generates the spoken gist and optional extra detail and returns them to you. Anthropic processes the descriptor only to produce that response.

The backend does not write page descriptors, their contents, or the model's replies to its logs. It keeps only minimal operational records (such as the time of a request, the response status, and a coarse request counter used to prevent abuse), which do not include the page content.

If the backend is unavailable or your daily quota is reached, AloudGuide uses an offline template summary generated inside the extension instead, with no network request.

What Is Stored

AloudGuide stores extension settings, such as whether the button is shown, button size, and premium/subscription state, using Chrome extension storage. For paid users it also stores a Premium license token, which the extension sends to the AloudGuide backend so the backend can confirm the subscription. The token does not contain personal browsing data.

AloudGuide also keeps a short-lived session cache of recent page explanations so clicking again on an unchanged page can replay the answer without another model request. The cache is cleared by the browser session and is not intended as a history log.

What Is Shared

AloudGuide shares page descriptors with Anthropic (as a processor that runs the Claude model) only when you ask AloudGuide to read a page. Anthropic acts on the descriptor solely to return the explanation. AloudGuide does not sell personal data and does not use it for advertising.

Scam warnings stay on your own device. When a page looks suspicious, AloudGuide includes a spoken caution in the explanation it reads aloud to you. It does not email, text, or send that warning to anyone, and there is no remote alert channel.

Your Choices

You can hide the AloudGuide button from the popup. You can also remove the extension from Chrome at any time.
